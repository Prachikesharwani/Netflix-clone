## Netflix-Clone

This is a Netflix clone featuring user authentication using Firebase, content browsing, and video streaming capabilities.
Implemented a dynamic user interface with seamless data management.


## Installation

```bash
  cd client
  npm Install
  npm start
  cd ..
  cd server
  npm Install
  npm start
```
